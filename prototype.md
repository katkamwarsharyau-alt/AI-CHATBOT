# 📱 Prototype Explanation

## 📌 Overview
The prototype is a working chatbot system for students.

## 💡 How it Works
- User enters question (example: exam timetable)
- Chatbot processes input
- Returns relevant answer

## 🖥 UI Description
- Login Page
- Chatbot Interface
- Response Display Area

## 🧪 Example

User: exam timetable  
Bot: Displays exam schedule  

User: placement  
Bot: Shows placement details
