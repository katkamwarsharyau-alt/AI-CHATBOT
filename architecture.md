# 🏗 System Architecture

## 📌 Overview
The system is built using Flask backend and HTML frontend.

## 🔄 Workflow

User Input → Web Interface (HTML/CSS)
→ Flask Backend (Python)
→ Processing Logic
→ Response to User

## 🧩 Components
- Frontend: HTML, CSS, JS
- Backend: Flask (Python)
- Database: SQLite

## 📊 Diagram (Text)
User → UI → Flask → Database → Response
