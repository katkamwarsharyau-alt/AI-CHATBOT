# 🎓 AI Academic Chatbot

## 📌 Project Overview
This project is an AI chatbot that helps students with academic and administrative queries like exam timetable and placement information.

## 🚀 Features
- Exam timetable
- Placement updates
- Student login system
- Chatbot interaction

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite Database

## ▶️ How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run:
   python app.py
